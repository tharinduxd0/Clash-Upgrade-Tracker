## üìù Usage

1.  **Select Village:** Use the dropdown at the top to select or create a new account (e.g., "TharinduXD").
2.  **Enter Time:** Input the upgrade duration (e.g., 14 Days, 12 Hours).

4.  **Start:** Click "Start Timer".
